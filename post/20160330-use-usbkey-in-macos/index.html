网银 U盾 Mac

工行和中行网银全部顺利支持Mac平台

Posted by [不一般的凡] on Wed, Mar 30, 2016

终于把 MBP 上的 Windows 删了，现在几乎所有工作都可以在 Mac 下完成，除了一样：网上银行。无奈再装个虚拟机好了，而在试验工行登录时，无意中发现有提示信息，有 Mac 下的 UsbKey 工具，转而登录中行时，竟也发现有 Mac 版 UsbKey 工具，赶紧安装上试验下，结果是：完全没问题！！！这下最后一点保留 Windows 的理由也没有了。这个世界终于清静了……
